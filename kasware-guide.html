KasWare EVM Setup Guide

1. Switch to IGRA Galleon in KasWare

Click KasWare icon → Select IGRA Galleon Testnet from network dropdown

2. Open KasWare Settings

Click the ⚙️ gear icon (Settings) → Look for EVM or Network settings

3. Enable EVM Support

Make sure these are enabled:

  • ☑️ Enable EVM Support
  • ☑️ Injected Provider
  • ☑️ Web3 Compatibility

4. If no settings, try this:

1. Open KasWare → Click network dropdown

2. Select IGRA Galleon

3. Then refresh this page

→ Test Again Here

Key: You MUST switch to IGRA Galleon in KasWare first before the EVM provider will inject!
